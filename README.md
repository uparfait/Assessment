

*** News managments system ***


this is a news app system whic allow admin to create,list,update and add news in a system 

*** Technologies used ***

- Nextjs with react
- Express js
- mongodb for database
- multer for images upload

*** how to run ***

open project directory
*** navigate to backend ***
type `npm install`
type `node main.js`
*** navigate to frontend ***
type `npm install`
type `npm run dev`

then backend and frontend will run as expected!

`backend run on http://localhost:2025`
`frontend may run on http://localhost:3000 due to nextjs server`
